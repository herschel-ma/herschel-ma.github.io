算法原理

  • 插入算法的思想是：将数据分为两部分，一部分已经排序，一部分未排序，遍历未排序的部分i,放在已排序的结尾，如果比已排序最后一个大，前移一位，直到位于符合的位置。

def insert_sort(alist):
    """插入排序"""
    n = len(alist)
    for j in range(1, n):
        i = j
        while i > 0:
            if alist[i] < alist[i-1]:
                alist[i], alist[i-1] = alist[i-1], alist[i]
                i -= 1
            else:
                break


if __name__ == "__main__":
    a = [93, 54, 77, 31, 55, 44, 226, 55]
    print(a)
    insert_sort(a)
    print(a)

def insert_sort(alist):
    n = len(alist)
    for j in range(1, n):
        for i in range(j, 0, -1):
            if alist[i] < alist[i-1]:
                alist[i], alist[i-1] = alist[i-1], alist[i]


if __name__ == "__main__":
    a = [93, 54, 77, 31, 55, 44, 226, 55]
    print(a)
    insert_sort(a)
    print(a)

实现解释

  • 其实这个算法的核心部分只是比较的部分而已，注意理解过程，分为两部分，第一部分是已排序序列，第二部分是未排序序列，从未排序的部分的索引循环遍历，与前面一个值比较，比他大不管（升序是这样），比他小则交换值的位置，注意这个比较过程是在循环中进行的，每一次从未排序的序列中遍历出来放在前面排序的序列尾部，再与前面的比较，所以索引每次减一。
  • 所以内部循环就控制上面说的减一，外部循环控制遍历。
