希尔排序

算法原理

  • 简单来说，希尔排序是基于插入排序的，先回顾一遍插入排序。
  • 插入排序是，假设两个序列，一个已经排好序，一个未排序序列，依次从未排序的序列中拿出一个值放在已经排序的序列的末尾，然后，如果这个数比前一个数小，交换位置，直到处于合适的位置。

def insert_sort(aList):
    """插入排序"""
    n = len(aList)
    for j in range(1, n):
        i = j
        while i > 0:
            if aList[i] < aList[i-1]:
                aList[i], aList[i-1] = aList[i-1], aList[i]
                i -= 1
            else:
                break
    return aList

if __name__ == "__main__":
    aList = [13, 14, 94, 33, 82, 25, 59, 94, 65, 23, 45, 27, 73, 25, 39, 10]
    response = insert_sort(aList)
    print(response)

  • shell_sort的算法原理，基于insert_sort, 只不过初始的i不同
  • 找一个初始的gap, 称之为间隙，比如[13, 14, 94, 33, 82, 25, 59, 94, 65, 23, 45, 27, 73, 25, 39, 10]， 比如去步长为5，那么取前5个为一个子序列，假设的，每次取子序列的第一个元素进行插入算法，说乱了，不会再自己查

def shell_sort(aList):
    n = len(aList) 
    gap = n//2  # 设置gap为长度一般开始
    # 里面的插入排序实现之后，就考虑外面的循环控制执行的次数，我们每次从gap开始，到n-1结束，注意我们操作的是下标
    # [gap, gap+1, ..., n-1]
    while gap > 0:
        for j in range(gap, n):
            i = j
            while i > 0:
                if aList[i] < aList[i-gap]:
                    aList[i], aList[i-gap] = aList[i-gap], aList[i]
                    i -= gap
                else:
                    break
        gap //= 2
    return aList


if __name__ == "__main__":
    aList = [13, 14, 94, 33, 82, 25, 59, 94, 65, 23, 45, 27, 73, 25, 39, 10]
    response = shell_sort(aList)
    print(response)
